# Scheme48

My implementation of [Write Yourself a Scheme](http://en.wikibooks.org/wiki/Write_Yourself_a_Scheme_in_48_Hours)

## Install
```bash
cabal sandbox init
cabal install
```

## Layout
* `src/` code for the actual interpreter

## Usage
```bash
cabal run
```
